Highlights

  • Built-in generators for GUID, number, email, phone, and string data
  • Optional clipboard history and clipboard restore
  • Light, dark, and system theme support

About MockPaste

MockPaste is a lightweight tray application for Windows 10+ built with .NET 10. It stays out of the way until you need it, then lets you generate and paste test data in a few keystrokes.

It is useful for QA, development, demos, and repetitive form-filling tasks where realistic placeholder values save time.
